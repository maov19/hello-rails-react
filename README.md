## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running follow these simple example steps.

### Prerequisites

In order to run this project you need:

- Code Editor (Vs Code)

- Terminal

- Install ruby on your computer

- Install rails on your computer

### Setup

Clone this repository by using the command line to your desired folder:  

```sh
git clone https://github.com/maov19/hello-rails-react.git

cd hello-rails-react
```

### Install

Install this project with:

```sh
cd hello-rails-react
 bundle install (install all RoR dependencies)
 yarn install (install React dependencies)
```

### Usage

To run the project, execute the following command:

```sh
rails db:create
rails db:migrate
rails db:seed
yarn build --watch
rails server
```

### Test

```sh
  rspec
```
